INSTRUÇÕES PARA EXECUTAR O PROGRAMA

1. Certifique-se que possui Java 8 instalado no computador, link para instalação abaixo
https://www.java.com/pt-BR/download/ie_manual.jsp?locale=pt_BR
2-Extrai a pasta compactada
3. Execute o arquivo "Clique_para_Executar.bat" com duplo clique
4. O programa deve abrir automaticamente

EM CASO DE PROBLEMAS:
- Verifique se o Java está instalado corretamente
- Execute o comando "java -version" no cmd para confirmar a versão
- Certifique-se que os arquivos .jar e .bat estão na mesma pasta

MODO DE USO
1-Primeiro passo seria criar um pedido, preenchendo o campo de comprimento, selecionando a altura desejada, a cor e se o pedido está confirmado ou não
3-Após isso, você deve selecionar o pedido na lista, assim podendo verificar os materiais necessários para construir a cerca vendida, Desenhar a cerca ou excluir ela da lista
(A cerca vendida pode possuir um tamanho diferente da cerca pedida, devido a fixação do tamanho da tela)

DETALHES
1-Limitei o tamanho máximo da cerca em 50 metros, para evitar problemas no desenho
2-A data do pedido aparecerá somente se "Confirmado" estiver marcado como "Sim"
3- A diferença entre o tamanho do pedido, e o tamanho efetivamente vendido só aparecerá quando houver diferença, e pode ser visualizado nos detalhes do pedido.
4- A lista está em ordem de inserção, pedidos mais recentes aparecerão no topo da lista

CONTEÚDO DO BAT
java -jar Executavel_Projeto_Weg.jar
